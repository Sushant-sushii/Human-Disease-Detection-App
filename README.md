# Jeevan AI - Human Disease Detection System

**Jeevan AI** is a modern web-based healthcare assistant designed to help users identify potential health issues based on biological symptoms. Built for a B.Tech CSE project, it demonstrates the integration of Large Language Models (LLMs) into functional web applications.

### 🚀 Key Features
* **AI-Powered Assessment:** Uses **Gemini 2.5 Flash** for high-speed, logical symptom mapping and health insights.
* **Body-Part Specific Logic:** Interactive UI allowing users to focus analysis on specific areas (Head, Chest, etc.).
* **Professional PDF Reports:** Generates clean, printable assessment reports styled for medical documentation.
* **Responsive Design:** Built with **Tailwind CSS v4.0**, featuring glassmorphism effects and mobile-first responsiveness.

### 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **AI Engine:** Google Gemini 2.5 Flash API
* **Frontend:** HTML5, Tailwind CSS v4.0 (Alpha/Beta), JavaScript
* **Deployment:** Vercel (Serverless Functions)
* **Data Handling:** Markdown-to-HTML processing for clean AI outputs

### 📦 Installation & Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Set your `GOOGLE_API_KEY` as an environment variable.
4. Run the development server: `python index.py`.
